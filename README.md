# twitterbot
Using twitter API to create tweets. Examples include using Tweepy and TwitterAPI modules. See [dottodot] (http://www.dototot.com/how-to-write-a-twitter-bot-with-python-and-tweepy/) for a quick introduction.

Many of the examples will not run, as is, and must be customized to a given user's environment. Some are works in progress and not yet completed.
